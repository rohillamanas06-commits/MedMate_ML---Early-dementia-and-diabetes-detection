## 🌟 Overview

MedMate ML is an AI-powered web application designed to predict health conditions such as dementia and diabetes using advanced machine learning models. Built with React, Python, and Tailwind CSS, MedMate ML empowers users to assess health risks and make informed decisions.

### ✨ Key Features

- **🧠 Dementia & Diabetes Prediction**: ML models trained on real datasets
- **⚡ Real-time Results**: Instant health risk analysis
- **📊 Detailed Reports**: Visualized prediction outcomes
- **🎨 Modern UI**: Responsive interface with Tailwind CSS
- **🔒 Secure**: No sensitive data stored
- **📱 Mobile Friendly**: Optimized for all devices

---

## 🚀 Tech Stack

### Frontend
- **React** - UI library
- **Vite** - Fast build tool
- **Tailwind CSS** - Utility-first CSS framework

### Backend
- **Python** - ML model training and inference
- **Flask (optional)** - API server (if used)
